# COMP490-CyberhuntGame

Cyberhunt is an educational cryptography game made with the intent of teaching players the following encryption algorithms: Caesar cipher, AES, RSA, and hashing. In our puzzle-RPG game, you play as an alien scraping an abandoned human laboratory for valuable information to aid your side in the inter-species war. 

### Build Instructions

***Our entire game has been exported and zipped into a YYZ file, located in the /artifacts folder.*** To build it to your local machine, you will first need to install the GameMaker IDE from the website here: 
https://gamemaker.io/en/download 
Follow the installation guide (you can choose to keep the default settings). Additional downloads may be required by the program upon starting GameMaker, but this will be handled by the IDE. If you are stuck at any stage of the process, follow this guide here:
https://gamemaker.io/en/help/articles/installing-gamemaker

Once you have GameMaker successfully installed into your machine, download and import the YYZ file into GameMaker (File > Import Project). Choose the directory where you want to house the project and GameMaker will extract the files for you. 

Please note that the YYP file and YYZ file are two different things! The YYP file contains metadata related to the project and will not open the game on its own without the accompanying resources!
